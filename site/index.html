Everything, in one window

Not a shortcut to installers.
An actual replacement for them.

Every action below runs a real command against winget or Chocolatey — there's no fake progress bar hiding a manual download somewhere.

One-Click Catalog

Browse 100+ apps across browsers, IDEs, databases, DevOps tools and more. Search, filter by category, favorite what you use.

Real Install Queue

Concurrent downloads with live speed and ETA. Pause, resume, cancel or retry any item — the queue survives a closed window.

Setup Profiles

Bundle your whole toolchain — "React Developer," "Data Science," your own mix — install it in one click, share it with a code.

Update Manager

Scans everything installed against winget/Chocolatey and shows exactly what's behind. Update one, a few, or all of it.

Uninstall Manager

Live-detects what's actually installed on your machine — not just what you installed through AIOforge — and lets you uninstall, repair or reinstall it.

PC Setup Tweaks

Dark mode, file extensions, Windows Terminal, PowerShell 7, WSL, Developer Mode — real registry edits and real installs, not decoration.

Backup & Restore

Snapshot your installed apps, profiles and settings with a checksum. Restore the whole thing on a fresh machine in one click.

Admin Deployment

Generate a PowerShell rollout script — and optionally the installers themselves — to set up a whole fleet of PCs the same way.

Built on Trust

Every install runs through winget or Chocolatey's own signature verification. AIOforge never runs unsigned code of its own.

How it works

Three steps. No wizard fatigue.

Download & install

Grab the installer, run it — it'll ask for admin rights once, since that's what real system-wide installs need — and AIOforge opens.

Pick your apps

Search the catalog, star your favorites, or grab a ready-made profile like "React Developer" or "Student Pack."

Let the queue work

Installs run in the background with real progress, speed and ETA. Close the window — the queue keeps going.
